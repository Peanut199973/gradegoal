GRADEGOAL - QUICK START GUIDE
==============================

Follow these steps to get your grade calculator online:

STEP 1: CREATE GITHUB REPOSITORY
---------------------------------
1. Go to https://github.com
2. Click the "+" icon â†’ "New repository"
3. Repository name: gradegoal
4. Make it Public
5. DON'T tick "Add a README file"
6. Click "Create repository"

STEP 2: EDIT package.json
--------------------------
1. Open the package.json file
2. Find this line:
   "homepage": "https://YOUR_GITHUB_USERNAME.github.io/gradegoal",
3. Replace YOUR_GITHUB_USERNAME with your actual GitHub username
4. Save the file

STEP 3: UPLOAD FILES TO GITHUB
-------------------------------
On your new empty repository page:
1. Click "uploading an existing file"
2. Drag ALL the files from this folder into the box
   (Make sure to include the hidden .gitignore file!)
3. Scroll down and click "Commit changes"

STEP 4: INSTALL NODE.JS (if you don't have it)
----------------------------------------------
1. Download from: https://nodejs.org/
2. Install it (choose all default options)
3. Restart your computer

STEP 5: DEPLOY THE APP
-----------------------
1. Open Terminal (Mac) or Command Prompt (Windows)
2. Navigate to this folder:
   cd path/to/gradegoal
3. Run these commands:
   npm install
   npm run deploy

4. Wait for it to finish (may take a few minutes)

STEP 6: ENABLE GITHUB PAGES
----------------------------
1. Go to your repository on GitHub
2. Click "Settings" (top menu)
3. Click "Pages" (left sidebar)
4. Under "Source": select "gh-pages" branch
5. Click "Save"

DONE! Your calculator is now live at:
https://YOUR_USERNAME.github.io/gradegoal

HOW TO USE GRADEGOAL:
---------------------
1. Set your target overall percentage
2. Configure how much Year 2 and Year 3 count (e.g. 40% and 60%)
3. Set semester weights for each year
4. Add modules (starts with 3 per semester, add/remove as needed)
5. Add assessments to each module (starts with 2 per module, add/remove)
6. Enter your scores as you get them
7. See what you need to achieve your target!

All your data saves automatically in your browser!

TO MAKE CHANGES LATER:
----------------------
1. Edit src/App.js (the main calculator code)
2. Test locally: npm start
3. Deploy changes: npm run deploy
4. Upload your changes to GitHub:
   git add .
   git commit -m "Updated calculator"
   git push

Need help? The README.md file has more detailed instructions!
